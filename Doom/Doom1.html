Dom? O que é isso?

O DOM (Document Object Model) é uma interface de programação para documentos HTML, XML e XHTML. Ele representa a estrutura do documento como uma árvore de objetos, onde cada elemento do documento é representado por um objeto no DOM. O DOM permite que os desenvolvedores acessem e manipulem os elementos, estilos, conteúdo e eventos do documento.
